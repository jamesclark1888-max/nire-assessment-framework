# Dimension 8: Round & Capital Strategy

**The VC underwriting question this answers:** *"Is this fundable now at sensible terms, with clear use of funds?"*

**Status:** 🟡 Skeleton. Questions to be drafted in v0.2.0.

---

## Why this dimension exists

Even a strong company can have an un-fundable round. Excessive valuations, unclear use of funds, weak existing investors, or messy cap tables can all kill an otherwise solid raise.

This dimension assesses the structural fundability of the current round.

---

## What this dimension assesses (planned)

1. **Round size**: appropriate for stage and milestones
2. **Valuation**: defensible against benchmarks
3. **Use of funds**: specific and milestone-linked
4. **Cap table health**: clean, well-structured
5. **Existing investor signal**: who's already in (or pre-committed)
6. **Runway**: how long the round funds
7. **Next-round positioning**: clear path to the milestones needed for the next raise

---

## Planned question structure

### Required tier (estimated 5-6 questions)
- Round size targeted
- Pre-money valuation
- Use of funds breakdown
- Months of runway from raise
- Existing committed amount (lead, soft-circles)
- Cap table health (clean or complications)

### Recommended tier (estimated 5-7 questions)
- Milestones to be hit with this raise
- Path to next round (Series A milestones, if seed)
- Existing investor list
- Prior funding history
- Convertible / SAFE conversion mechanics
- Option pool status

### Optional deep-dive (estimated 3-4 questions)
- Cap table upload
- Detailed budget upload
- Term sheet template
- Investor presentation upload

---

## Heavy regional and stage variation

Round sizes and valuations differ substantially by region (UK seed median £1-1.5m vs US $2-3m). Benchmarks are regional. Stage adaptation is heavy.
